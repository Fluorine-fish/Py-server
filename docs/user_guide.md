# 用户指南

## 系统概述
该系统是一款基于视觉分析的儿童智能台灯管理系统，通过机械臂实现智能照明和姿势提醒功能。系统针对儿童写作业、阅读等场景进行了优化，为家长提供全面的监控和管理功能。

## 功能说明

### 1. 视觉分析功能
- 多关节姿势检测和分析
- 用眼距离监测
- 光线环境评估
- 面部表情识别

### 2. 提醒功能
- 久坐提醒
- 不良姿势提醒
- 光线不足提醒
- 休息建议提醒

### 3. 照明控制功能
- 色温调节（3000-6500K）
- 亮度调节（0-100%）
- 照明模式选择
  - 手动模式
  - 自动模式
  - 阅读模式
  - 休息模式

### 4. 数据统计功能
- 用眼时间统计
- 坐姿分析报告
- 情绪状态跟踪
- 使用习惯分析

## 使用指南

### 1. 系统登录
1. 打开浏览器访问系统
2. 进入主控制面板

### 2. 基础设置
1. 设备连接：
   - 点击"设备管理"
   - 选择正确的串口设备
   - 设置通信波特率
   - 点击"连接"按钮

2. 参数配置：
   - 设置提醒时间阈值
   - 调整检测灵敏度
   - 配置光线参数
   - 保存设置

### 3. 实时监控
- 查看实时视频流
- 监控姿势状态
- 观察光线环境
- 查看情绪分析

### 4. 照明控制
1. 手动控制：
   - 调节亮度滑块（0-100%）
   - 调节色温滑块（3000-6500K）
   - 选择照明模式

2. 自动模式：
   - 系统根据环境自动调节
   - 可设置自动调节范围
   - 支持定时切换

### 5. 数据查看
1. 历史记录：
   - 查看历史提醒记录
   - 浏览坐姿数据
   - 导出统计报告

2. 数据分析：
   - 查看趋势图表
   - 分析使用习惯
   - 生成分析报告

# 用户指南

## 系统概述
该系统是一款基于视觉分析的儿童智能台灯管理系统，通过机械臂实现智能照明和姿势提醒功能。系统针对儿童写作业、阅读等场景进行了优化，为家长提供全面的监控和管理功能。

## 功能说明

### 1. 视觉分析功能
- 实时姿势检测
- 用眼距离监测
- 情绪识别分析
- 光线环境检测

### 2. 提醒功能
- 久坐提醒：当检测到连续久坐超过预设时间时，系统会通过机械臂发出提醒
- 坐姿提醒：检测到不良坐姿时，系统自动提醒并记录
- 距离提醒：当与屏幕/书本距离不适时发出提醒
- 光线提醒：环境光线不足时进行提示

### 3. 数据统计功能
- 用眼时间统计
- 坐姿分析报告
- 情绪状态跟踪
- 使用习惯分析

## 使用指南

### 1. 系统登录
1. 打开浏览器访问系统地址
2. 使用管理员账户登录
3. 进入主控制面板

### 2. 基础设置
1. 设备连接：
   - 点击"设备管理"
   - 选择正确的串口设备
   - 设置通信波特率（默认115200）
   - 点击"连接"按钮

2. 参数配置：
   - 设置提醒时间阈值
   - 调整检测灵敏度
   - 配置光线参数
   - 保存设置

### 3. 实时监控
1. 视频监控：
   - 查看实时摄像头画面
   - 观察姿态检测结果
   - 监控距离提示

2. 状态监控：
   - 检查设备连接状态
   - 查看当前检测结果
   - 观察提醒状态

### 4. 提醒功能使用
1. 久坐提醒：
   - 开启自动提醒功能
   - 设置提醒时间间隔（默认30分钟）
   - 可选择提醒方式

2. 坐姿提醒：
   - 启用坐姿检测
   - 设置灵敏度阈值
   - 配置提醒模式

### 5. 数据查看
1. 历史记录：
   - 查看历史提醒记录
   - 浏览坐姿数据
   - 导出统计报告

2. 数据分析：
   - 查看趋势图表
   - 分析使用习惯
   - 生成分析报告

## 照明控制系统

### 访问照明控制界面
1. 打开Web界面，点击导航栏中的"照明控制"选项
2. 或直接访问 `/new-ui/lighting` 路径

### 调节照明参数

#### 亮度控制
- 使用亮度滑块调节灯光亮度（0-100%）
- 实时预览区域会显示当前亮度效果
- 亮度数值会实时显示在滑块右侧

#### 色温控制
- 使用色温滑块调节灯光色温（2700-6500K）
- 较低色温（2700K）呈现温暖黄光
- 较高色温（6500K）呈现冷白光
- 色温数值会实时显示在滑块右侧

### 照明模式

系统提供四种照明模式：

1. 手动模式
   - 完全手动控制亮度和色温
   - 适合个性化照明需求

2. 自动模式
   - 根据环境光自动调节
   - 滑块将被禁用
   - 系统会根据检测到的光线条件自动调整

3. 阅读模式
   - 预设适合阅读的照明参数
   - 中等亮度（约70%）
   - 适中色温（约4000K）

4. 休息模式
   - 柔和的照明效果
   - 较低亮度（约30%）
   - 温暖色温（约3000K）

### 应用设置

1. 调整参数后，点击"应用设置"按钮使更改生效
2. 系统会立即将设置发送到照明设备
3. 成功应用后会显示确认提示
4. 如果发生错误，会显示错误信息

### 注意事项

1. 确保串口连接正常
   - 在使用照明控制功能前，请确保串口已正确连接
   - 可以在串口控制页面查看连接状态

2. 模式切换
   - 切换到自动模式时，手动控制会被禁用
   - 从自动模式切换回其他模式时，会恢复之前的设置

3. 参数范围
   - 亮度：0-100%
   - 色温：2700-6500K
   - 超出范围的值会被自动限制在有效范围内

4. 错误处理
   - 如果设置失败，请检查串口连接
   - 可以尝试断开并重新连接串口
   - 如果问题持续，请查看系统日志获取详细信息

### 常见问题解答

1. 为什么无法调节亮度和色温？
   - 检查是否处于自动模式
   - 确认串口连接状态
   - 验证设备是否正常响应

2. 照明设置没有立即生效？
   - 确保点击了"应用设置"按钮
   - 检查串口连接状态
   - 查看是否有错误提示

3. 如何恢复默认设置？
   - 选择手动模式
   - 将亮度设置为50%
   - 将色温设置为4000K
   - 点击"应用设置"

4. 自动模式不工作？
   - 确认环境光传感器工作正常
   - 检查系统日志中是否有相关错误
   - 尝试切换回手动模式再切换到自动模式

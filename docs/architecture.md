# 系统架构文档

## 整体架构

该系统采用前后端分离的Web架构，主要包含以下几个核心模块：

### 1. 前端模块
- Web界面（HTML/CSS/JavaScript）
- 实时数据可视化
- 用户交互界面
- 响应式布局设计

### 2. 后端服务
- Flask Web服务器
- 视频流处理模块
- 串口通信模块
- 数据库管理模块

### 3. 功能模块
- 坐姿检测模块
- 情绪分析模块
- 视力保护模块
- 机械臂控制模块

### 4. 数据存储
- SQLite数据库
- 文件系统存储
- 缓存系统

## 模块详细说明

### Web服务器 (app.py)
- 处理HTTP请求
- 路由管理
- 会话管理
- 错误处理

### 视频流处理 (modules/video_stream_module.py)
- 摄像头数据采集
- 图像预处理
- 实时视频流推送
- 帧率控制

### 串口通信 (serial_handler.py)
- 串口连接管理
- 数据帧处理
- 错误检测和恢复
- 设备状态监控

### 数据库管理 (db_handler.py)
- 数据模型定义
- 数据存取操作
- 查询优化
- 数据备份

### 分析模块
- 姿态检测算法
- 情绪识别算法
- 专注度分析
- 视力保护算法

## 数据流

1. 数据采集流程
   ```
   摄像头 -> 视频流模块 -> 图像处理 -> 分析模块 -> 数据库
   ```

2. 控制流程
   ```
   Web界面 -> 后端服务 -> 串口模块 -> 机械臂
   ```

3. 数据展示流程
   ```
   数据库 -> Web API -> 前端展示 -> 用户界面
   ```

# 新页面整合计划

## 产品前提：

- **产品前提整条内容不允许Agent修改！！**

- 这是一款儿童智能台灯的家长端网页，家长可以利用网页查看台灯从视觉相机分析得到的各种信息，所有的查看功能供家长使用，所有的分析功能均针对孩子，不监看电脑等电子产品的用眼状况，而是写作业阅读的用眼状况。
- 这款智能台灯是以四轴机械臂为核心，台灯作为末端执行器，请务必完成以下的功能：
   1. 久坐提醒：检测到孩子久坐到一定时间，通过串口向机械臂发送需要需要提醒的信息（请在原有协议的未使用的帧字节中改动）
   2. 坐姿提醒：请适当优化坐姿检测算法（现在只有检测连线夹角）可以采用多个关节的综合检测，在孩子不良坐姿时通过串口向机械臂发送需要需要提醒的信息（请在原有协议的未使用的帧字节中改动），并且在后端做相应记录（依照下文的功能）

- **产品前提整条内容不允许Agent修改！！**

## 后端开发任务

1. 数据模型设计
   ```python
   # 坐姿记录表结构示例
   class PostureRecord(db.Model):
       id = db.Column(db.Integer, primary_key=True)
       timestamp = db.Column(db.DateTime, default=datetime.utcnow)
       posture_quality = db.Column(db.String(20))  # good, slightly_bad, bad, severe
       posture_angle = db.Column(db.Float)
       duration = db.Column(db.Integer)  # duration in seconds
       
   # 情绪记录表结构示例
   class EmotionRecord(db.Model):
       id = db.Column(db.Integer, primary_key=True)
       timestamp = db.Column(db.DateTime, default=datetime.utcnow)
       emotion_type = db.Column(db.String(20))  # happy, sad, angry, focused, confused
       confidence = db.Column(db.Float)
       
   # 专注度记录表结构示例
   class FocusRecord(db.Model):
       id = db.Column(db.Integer, primary_key=True)
       timestamp = db.Column(db.DateTime, default=datetime.utcnow)
       focus_score = db.Column(db.Float)  # 0-100
       duration = db.Column(db.Integer)  # duration in seconds
   ```
